Useful Bash Scripts
===

* `sed_dir` - Replace text for all files given a directory
  * Takes 3 params:
    * `$1` - Directory
    * `$2` - Term to change
    * `$3` - Replacement term

* `add_to_git_push_queue` - Adds a git push to do tomorrow
  * Takes 2 params
    * `$1` - Directory
    * `$2` - Commit message

* `git_push` - Adds, commits, and pushes to github
  * Takes 2 params
    * `$1` - Directory
    * `$2` - Commit message
